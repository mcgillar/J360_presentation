CSS stands for Cascading Style Sheets.

There are two ways to link your HTML documents to stylesheets, which can be internal or external. Internal styles are good for beginners because often there is not a whole lot of CSS being used. But generally, when you start becoming more advanced with html, and css, it's more common to link your HTML to an external stylesheet. The reason for this is that it saves time and space, by linking to external stylesheets, you can create a single CSS document that styles endless pages of html. Example

Tips and Tricks See what you can do.

Borders Style, width, color, individual sides, radius.

Margins By individual side; auto, inherit.

Padding Individual sides vs. shorthand.

Outline Similar to borders; goes outside; specify style, color and width.

Text Color, alignment, decoration, transformation, fonts etc.

Colors Can be referenced by RGB, Hexadecimal, or by name.

Backgrounds Color and image backgrounds; background-position.

Height/Width Percentage vs. pixel; maximum and minimum.

Links Visited, hover, active.

Lists Un-ordered vs. ordered lists; list markers.

Tables Borders, width & height, horixontal & vertical alignment, padding.

Opacity Transparency; hover transparency.

NavBar Vertical & horizontal navigation bars; full height; floating.

Dropdowns You can drop down basic text, a menu, or even an image.

Tooltips Basic tooltip positioning; animated on hover.

Galleries Fixed galleries; auto vs. manual slideshows.
